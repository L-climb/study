# 深度学习基础知识速查（高频必会版）

如果你已经会 PyTorch，那么接下来最重要的是理解：

```text
数据
 ↓
特征
 ↓
模型
 ↓
损失函数
 ↓
反向传播
 ↓
梯度下降
 ↓
模型更新
````

这就是所有深度学习模型训练的本质。

---

# 1 什么是深度学习

本质：

```text
输入 x
 ↓
神经网络 f(x)
 ↓
输出 y
```

例如：

```text
图片
 ↓
CNN
 ↓
猫
```

```text
文本
 ↓
Transformer
 ↓
情感分类
```

---

# 2 神经元（Neuron）

最小计算单元：

公式：

y = wx + b

其中：

```text
w = 权重
b = 偏置
x = 输入
```

例如：

```python
y = 2*x + 1
```

---

# 3 激活函数

如果没有激活函数：

```text
无论多少层
=
一层线性模型
```

所以必须引入非线性。

---

## ReLU（最常用）

公式：

```text
f(x)=max(0,x)
```

图像：

```text
x<0 => 0
x>0 => x
```

优点：

```text
计算快
梯度稳定
```

---

## Sigmoid

公式：

```text
1/(1+e^-x)
```

输出：

```text
0~1
```

常用于：

```text
二分类输出层
```

缺点：

```text
梯度消失
```

---

## Tanh

输出：

```text
-1~1
```

比Sigmoid稍好。

---

## GELU

Transformer默认激活函数。

GPT系列大量使用。

---

# 4 前向传播（Forward）

输入：

```text
x
```

经过网络：

```text
x
 ↓
Linear
 ↓
ReLU
 ↓
Linear
 ↓
输出
```

得到预测值：

```text
ŷ
```

---

# 5 损失函数（Loss）

衡量：

```text
预测值
和
真实值
差多少
```

---

## 回归

MSE

公式：

```text
(pred-target)^2
```

---

## 分类

CrossEntropy

最常用分类损失。

---

# 6 梯度（Gradient）

梯度：

```text
损失变化最快方向
```

数学上：

```text
∂Loss/∂W
```

表示：

```text
参数变化一点点
Loss变化多少
```

---

# 7 反向传播（Backpropagation）

核心思想：

```text
Loss
 ↓
输出层
 ↓
隐藏层
 ↓
输入层
```

逐层计算梯度。

PyTorch：

```python
loss.backward()
```

自动完成。

---

# 8 梯度下降（Gradient Descent）

目标：

```text
让Loss越来越小
```

更新公式：

```text
W = W - lr * gradient
```

其中：

```text
lr
=
learning rate
=
学习率
```

---

# 9 学习率（Learning Rate）

非常重要。

---

学习率太大：

```text
Loss震荡

甚至发散
```

---

学习率太小：

```text
训练特别慢
```

---

常见：

```python
1e-3
1e-4
5e-5
```

---

# 10 Epoch、Batch、Iteration

面试必问。

---

## Epoch

全部数据训练一次。

例如：

```text
10000条数据
全部跑完

=
1 epoch
```

---

## Batch

每次喂给模型的数据量。

例如：

```text
batch_size=32
```

一次处理：

```text
32条数据
```

---

## Iteration

一次参数更新。

例如：

```text
10000条数据

batch=100
```

则：

```text
100次iteration
=
1 epoch
```

---

# 11 过拟合（Overfitting）

表现：

```text
训练集准确率很高

测试集很差
```

说明：

```text
背答案
```

而不是学规律。

---

解决方法：

## Dropout

随机关闭神经元。

```python
nn.Dropout(0.5)
```

---

## 数据增强

图片：

```text
翻转

裁剪

旋转
```

---

## 增加数据量

最有效。

---

# 12 欠拟合（Underfitting）

表现：

```text
训练集很差

测试集也差
```

说明：

```text
模型太弱
```

---

解决：

```text
增加层数

增加训练时间

增加特征
```

---

# 13 Batch Normalization

简称：

```text
BN
```

作用：

```text
稳定训练

加快收敛
```

PyTorch：

```python
nn.BatchNorm2d()
```

---

# 14 Dropout

训练时：

随机删除部分神经元。

例如：

```python
nn.Dropout(0.5)
```

含义：

```text
50%概率失活
```

# 15 常见评价指标

## 分类

### Accuracy

准确率

```text
预测正确数
/
总数
```

---

### Precision

查准率

```text
预测为正
有多少真的正
```

---

### Recall

召回率

```text
真实为正
找回来多少
```

---

### F1

公式：

```text
2PR/(P+R)
```

综合考虑：

```text
Precision

Recall
```

---

## 二分类

### ROC-AUC

范围：

```text
0.5 ~ 1
```

越大越好。

---

## 目标检测

### IoU

公式：

```text
交集
/
并集
```

例如：

```text
预测框
和
真实框
```

重叠程度。

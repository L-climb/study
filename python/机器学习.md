# 17. PyTorch训练流程（必须背）

```python
for epoch in range(epochs):

    for x, y in loader:

        pred = model(x)

        loss = criterion(pred, y)

        optimizer.zero_grad()

        loss.backward()

        optimizer.step()
```

理解：

```text
预测

↓

计算Loss

↓

清空梯度

↓

反向传播

↓

更新参数
```
